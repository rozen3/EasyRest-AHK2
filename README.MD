## English

A lightweight, responsive, and modern-looking rest reminder for Windows, designed to mitigate work fatigue through regular forced or gentle reminders.

### âœ¨ Key Features
* **Dynamic Tray Icon**: Displays a real-time progress bar for work time and remaining minutes.
* **Multi-Theme Support**: Includes "Minimalist White", "Geek Black", and "Fresh Green" themes.
* **Interactive Transparency**: Adjust window transparency (0-100%) via a live slider.
* **Session Awareness**: Automatically pauses the timer when the system is locked or suspended.
* **Persistence**: Remembers window position, theme, work intervals, and transparency.
* **Forced Rest Mode**: Full-screen overlay with countdown. Press `Esc` for emergency exit.

### ğŸš€ Quick Start
1.  Ensure [AutoHotkey v2.0+](https://www.autohotkey.com/) is installed.
2.  Download and run `RestReminder.ahk`.
3.  Right-click the tray icon to customize your settings.

### ğŸ›  Configuration
All settings are automatically saved to `config.ini` in the script directory. No script editing required.
